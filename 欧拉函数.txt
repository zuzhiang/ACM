在数论，对正整数n，欧拉函数是小于等于n的数中与n互质的数的数目。 例如φ(8)=4，因为1,3,5,7均和8互质。

φ(n) = n*(1-1/p1)*(1-1/p2)*......(1-1/pn)   其中(p1.....pn)为N的素因子

欧拉函数公式：
性质① m是素数时，有φ(m)=m-1
性质② 当m、n互素时，φ(m*n)=φ(m)*φ(n)  （互素：最大公约数为1）
性质③ 对一切正整数n，有φ(p^n)=[p^(n-1)]*(p-1)

欧拉函数代码实现：
int euler(int n)
{
    int ret=1,i;
    for(i=2;i*i<=n;i++)
    {
        if(n%i==0)
        {
            n=n/i;
	    ret*=i-1;
            while(n%i==0)
	   { 
	        n=n/i;
		ret*=i;
	    }		
        }
    }
    if(n>1) ret*=n-1;
    return ret;
}
